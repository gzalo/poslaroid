# Poslaroid

Aplicación para tomar fotos e imprimirlas con impresoras térmicas conectadas por bluetooth!

Ahora con un poco de GenAi para convertir el estilo antes de imprimirlas. Usa el modelo [Flux2 klein (open weights)](https://bfl.ai/models/flux-2-klein).

Para usar esto, es necesario tener corriendo [ComfyUI](https://www.comfy.org/) con los modelos necesarios descargados, y el servicio de `genai/comfyui_img2img.py`. Al abrir la app aparecerá un campo adicional para especificar la URL del servicio.
